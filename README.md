<h1 align="center"> 🎲 Sorteador 🎲 </h1><br>

Aplicação web desenvolvida com <strong>HTML</strong>, <strong>CSS</strong> e <strong>JavaScript</strong> que permite ao usuário gerar um número aleatório dentro de um intervalo definido.

O usuário informa um valor mínimo e um valor máximo, e o sistema retorna automaticamente um número sorteado dentro desse intervalo.

<h3> Funcionalidades:</h3>

- **Definição de intervalo personalizado:** o usuário pode escolher qualquer número mínimo e máximo para o sorteio.
- **Geração de número aleatório:** o sistema utiliza lógica em JavaScript para gerar um número aleatório dentro do intervalo informado.
- **Interação simples e direta:** o resultado é exibido de forma imediata na tela, proporcionando uma experiência rápida e intuitiva.

<h3> Objetivo do projeto </h3>

- Manipulação do DOM;
- Captura de dados do usuário;
- Lógica de programação com JavaScript;
- Geração de números aleatórios;
